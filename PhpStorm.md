# PHPSTORM Keybard Shortcuts


```bash

# multiple cursors
Press Ctrl (Windows or Linux)/Alt(macOS) twice, and then without releasing it, press up or down arrow keys.

# next selection
alt + j

# toggle upper/lower case
ctrl + shift + u

# move clas to a separate file
click on class name and press F6


# suggests to import the namespace where it has detected the declaration of the class
Click on the class name then press Alt+Enter

# Reformat block of code
Code > Reformat Code, or press Ctrl + Alt + L

# reference
# - https://www.jetbrains.com/help/rider/Multicursor.html
```
